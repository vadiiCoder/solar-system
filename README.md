## 🚀 Overview

This is a **highly animated cosmic-themed portfolio website** built using **HTML, CSS, and JavaScript**.  
It presents the solar system in a **modern cinematic UI experience** with glowing planets, animated starfield background, and smooth transitions.

It is designed as a **creative portfolio project** to demonstrate:
- Advanced frontend UI design
- Animation systems
- Visual storytelling
- Responsive web development

---

## ✨ Features

🌠 Procedural animated starfield background  
🪐 Interactive solar system planet cards  
🎨 Advanced gradient + glow UI design  
⚡ Smooth fade & hover animations  
📱 Fully responsive layout (mobile + desktop)  
🎥 Integrated video showcase section  
🧠 Clean, structured, modern UI/UX design  

---

## 🧑‍🚀 Project Theme

The website takes users on a **journey through space**, exploring:

- ☿ Mercury — The scorched world  
- ♀ Venus — Earth's fiery twin  
- 🌍 Earth — The blue planet  
- 🔴 Mars — The red frontier  
- 🪐 Jupiter — The gas giant  
- 💍 Saturn — The ringed wonder  
- 🧊 Uranus — The tilted ice world  
- 🌊 Neptune — The stormy edge of the solar system
